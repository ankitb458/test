1. Unzip mylinkorder.zip
2. Move mylinkorder.php to /wp-content/plugins/
3. Activate the My Link Order plugin on the Plugins menu
4. Go to the "My Link Order" tab under Links and specify your desired
   order for link categories and links in each category
5. Modify sidebar template to use correct filter(additional parameter seperated by ampersands):
	wp_list_bookmarks('orderby=order&category_orderby=order');
